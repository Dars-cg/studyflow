# 📚 StudyFlow

StudyFlow es una aplicación web para la gestión de tareas y proyectos personales, desarrollada como proyecto de aprendizaje Full Stack.

El objetivo principal es fortalecer conocimientos en desarrollo web mediante la construcción de una aplicación moderna utilizando buenas prácticas de desarrollo, control de versiones y trabajo colaborativo.

---

# 🚀 Objetivos

- Aprender desarrollo Full Stack desde cero.
- Aplicar buenas prácticas de desarrollo de software.
- Comprender el funcionamiento de Git y GitHub.
- Trabajar de manera colaborativa utilizando Pull Requests.
- Desarrollar una aplicación web escalable.
